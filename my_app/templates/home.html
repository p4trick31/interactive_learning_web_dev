🎓 About the System

Educational Web-Based Game for Learning Web Development Programming Fundamentals

The Educational Web-Based Game is an interactive learning platform developed for the Dr. Emilio B. Espinosa Sr. Memorial State College of Agriculture and Technology (DEBESMSCAT). It is designed to provide an engaging and gamified learning experience for BSCS, BSIS, and BSCE students studying web development fundamentals.

The system combines tutorials, coding activities, quizzes, and game-based challenges to make learning more interactive and enjoyable. Students can track progress, earn rewards, compete on leaderboards, and receive automated feedback while mastering core web concepts.
